# 넥서스(NEXUS) 서비스 소개서

**AI 기반 기사 작성 업무 지원 시스템** | 작성일: 2025년 12월

---

## 1. 서비스 개요

서울경제신문 기자 228명을 대상으로 2025년 10월부터 운영 중인 AI 기사 작성 지원 시스템이다. 6개 카테고리, 12개 서비스로 구성되어 있으며, 현재 MVP 단계로 운영 중이다.

---

## 2. 문제 정의

### 2-1. 기자 업무의 병목

기자의 본업은 취재와 판단이다. 그러나 초안 작성, 교열, 제목 고민 등 반복 작업에 상당한 시간이 소요된다.

### 2-2. 기존 AI 활용의 한계

- **복잡한 로그인**: 5단계 이상의 인증 절차로 업무 흐름 단절 (예: 사내 로그인 → 외부 서비스 로그인 → 이메일 인증 → 보안 링크 클릭 → 2FA 인증)
- **프로젝트 탐색 어려움**: 300개 이상의 프로젝트에서 원하는 도구 찾기 어려움 (예: 팀 계정 선택 → 프로젝트 메뉴 → 검색 → 확인)
- **스타일 미반영**: 언론사별 제목 규칙, 표기법, 전문 용어 미반영 (예: 숫자 표기법, 외래어 표기, 인용 부호 규칙 등)

### 2-3. 프롬프트 복잡도 증가

AI 결과물의 품질을 높이기 위해 각 서비스별 프롬프트 규모가 증가했다. 2024년 9월 기준 3만 자(파일 4개)에서 2025년 9월 기준 10만 자(파일 14개)로 증가했다.

---

## 3. 넥서스가 해결하려는 것

- **복잡한 로그인** → 한 번의 로그인으로 모든 서비스 접근
- **프로젝트 탐색 어려움** → 직관적인 UI로 도구 즉시 선택
- **스타일 미반영** → 기자 전문 표현, 서술 구조, 논리 구조, 언론사별 스타일 반영
- **AI 지식 단절** → 관리자가 실시간으로 프롬프트 수정 가능 (예: 신조어, 인물 직책 변경, 최신 이슈 반영)

**설계 원칙**: Human-in-the-loop. AI는 선택지를 제시하고, 기자가 최종 판단한다.

## 5. 서비스 구성

- **버디** (일보/기사): 아이디어 입력 → 기사 방향 제안 → 초안 생성 (예: "홍보대행사" 입력 → 4개 방향 제시)
- **보도자료** (기업/공공): 보도자료를 기사 형식으로 변환 (예: 원문 50~80% 압축, 정치인 실명 필터링)
- **외신** (영어/일어): 외국어 기사를 한국 독자용 기사로 변환 (예: 출처 자동 표기, 번역투 제거)
- **퇴고** (단문/장문): 기사 구조와 문장 개선 (예: 첫 문장 70자 → 35자 압축)
- **제목** (생성/창의): 5~7가지 제목 옵션 제공 (예: 저널리즘형, 클릭형, 플랫폼별 추천)
- **교열** (경제/사회): 맞춤법, 문법, 전문용어 점검 (예: 금액 표기, 기업명, 반도체 용어 검토)

## 6. 사용자 피드백

**긍정**
- 로그인 한 번으로 모든 AI 도구를 사용할 수 있어서 편리하다
- 여러 선택지 중에서 선택할 수 있어서 좋다
- 제목 작성 시간이 단축되었다

**개선**
- 기자 본인의 채팅 기록을 AI가 학습하여 맞춤형 답변을 제공했으면 좋겠다
- 외신 서비스에서 링크만 붙여넣으면 자동으로 내용을 가져왔으면 좋겠다 (현재는 사용자가 직접 텍스트를 복사해야 함)
